## Operadores de comparação
  == -> igual
  === -> idêntico
  != -> diferente
  !== -> diferente exato
  > -> maior que
  >= -> maior ou igual
  < -> menor que
  <= -> menor ou igual

## Diferença entre '=' e '=='
  O '=' é o operador de atribuição(também chamado de recebe). Não confunda!
  Quando queremos saber se 1+1 é igual à 2 usamos o '=='.

## Operadores lógicos
  && -> operador AND(E)
  || -> operador OR(OU)
  ! -> operador NOT(Não)

## Operadores de tipo
  typeof -> usado para saber o tipo de uma variável
